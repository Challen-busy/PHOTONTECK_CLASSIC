## 2. ER 关系总图（核心 FK 链）

```
                       company (6 租户, company_id 贯穿全部业务实体)
                          │
customer ──< opportunity ──< quote ──< sales_order ──< shipment_request ──< outbound_order >── inventory_lot
   │            │                          │                                      │              │
 contact     product_line               sales_invoice ──> accounts_receivable    pick_line     inventory_movement
   │            │ (1线=1供应商)             │ (推金蝶)        │ (只读视图)            (拣批次)        (事件溯源)
customer_qualification                   special_batch_shipment(可隐藏)
   └─ cosign_line (并行会签)

supplier ──< purchase_order ──< inbound_receipt ──< inventory_lot   ──> customs_declaration (按 PO×型号×HS 聚合明细)
   │  (default_pa)  │  │             │ (inbound_line=批次)                │
 supplier_inquiry   │  purchase_notice                                customs_license
   │                purchase_invoice ──> accounts_payable             退运180天 → origin_declaration_id(自引用)
 stock_up_request(会签) / sample_sdn / rma / purchase_in_transit      shipment_tracking(顺丰,挂 出/入库/报关 运单号)

material(SN/LOT 管控) ──< product_code(型号×供应商) ；  location(三级库位) ── inventory_lot

★ 推送：PO/采购入库/SO/销售出库/销项发票/进项发票/库存调整/付款/预收预付 ── @effect ──> kingdee_outbox ──> 金蝶(OpenAPI)
★ 拉取：金蝶已确定应收应付视图 ──> kingdee_ar/ap_snapshot(只读展示)
★ 写动作 ── 审计四本(command_log/workflow_log/agent_log/workflow_def_audit, 只增)
```

> 关键不变量：① 所有业务单据带 `company_id`，**财务绝不合账**，一笔生意不跨 2 家公司；② 库存=`inventory_movement` 事件累加（`inventory_lot` 是投影）；③ 串货隔离=`inventory_lot.reported_customer_id` 出库校验；④ 业务单号=金蝶单据编号（推送幂等键）。
